## 概览

We use *mkdocs* to manage our documentation

## 贡献力量

We're always happy to see your PRs!

1) 安装依赖关系

```pip install mkdocs```

2) 构建
```
git clone --recursive https://github.com/emlid/navio2-docs
cd navio2-docs
mkdocs serve -a 0.0.0.0:8000
```
3) 做出修改

4) 测试你的改动

- 启动浏览器
- 在地址栏里输入localhost:8000
